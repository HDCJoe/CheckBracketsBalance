**App:**    CheckBracketsBalance

Checks that the number of opening brackets, parenthesis and braces match the number of closing.

#How to Use
- Create a file named input.txt in the same directory as the CheckBracketsBalance.py application
- Enter or paste the expression you want checked into the file
- Save the file
- Open a command prompt in this location
- Enter: python.exe .\CheckBracketsBalance.py
- If the brackets, parenthesis or braces are mismatched the application will report this and indicate where the issue is
